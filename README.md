# Tickets Pro (Next.js)

Proyecto migrado de Vite a Next.js usando App Router y TypeScript.

## Scripts

- `npm run dev`: inicia el servidor de desarrollo.
- `npm run build`: genera el build de produccion.
- `npm run start`: ejecuta la app en modo produccion.
- `npm run lint`: ejecuta lint con reglas de Next.js.

## Estructura principal

- `src/app`: rutas y layout de Next.js.
- `src/components`: componentes reutilizables.
- `src/pages`: vistas internas reutilizadas por las rutas del `app router`.
- `src/scss`: estilos SCSS globales y variables base.

## Rutas migradas

- `/` Dashboard
- `/tickets`
- `/users`
- `/settings`
- `/view_kanban`
